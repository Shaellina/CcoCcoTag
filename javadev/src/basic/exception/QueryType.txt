Dynamic Query vs Parameter Query

/**********************************************************/
/**Dynamic Query로 작성 할 경우 SQL Injection에 매우 위험**/
/**********************************************************/

1. Parameter Query로 작성이 매우 간결
2. 성능이 매우 좋음
 ( 좋을 수 밖에 없는 이유? -> query 를 실행하는 순서 상에서 같은 쿼리를 보내게 됨, 그래서 코드의 재사용으로 성능 향상 )
3. SQL Injection에 강함
